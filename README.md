# Read Me First

This is a Java application, that provides RESTful services, using Spring Boot.
This is just an example and should not be deployed to a Production environment.


# Getting Started

* Run the application using Maven: mvn clean spring-boot:run
* Swagger is available for this application, it can be used for testing. Once the application is running in your local environment, access [this link](http://localhost:8080/example/swagger-ui/index.html)
